# 爱弥斯桌宠 (Aemeath-AI)

一个可爱的虚拟桌面宠物应用，使用 Electron + Vue3 + TypeScript 开发。

## 功能特性

### 核心功能
- ✨ **透明窗口** - 无边框、透明背景，完美融入桌面
- 🎭 **角色动画** - 可爱的宠物形象和动画效果
- 🖱️ **拖拽移动** - 鼠标拖动窗口到任意位置
- 💾 **本地存储** - 数据自动保存到本地 JSON 文件

### 养成系统
- 🍖 **投喂玩法** - 5 种食物可选，不同效果
- 😊 **状态系统** - 饱食度、好感度、能量三维属性
- 📊 **等级成长** - 喂食获得经验，自动升级
- 🎨 **视觉反馈** - 爱心动画、数值提示、成功消息

### 交互体验
- 🎮 **右键菜单** - 快捷操作菜单
- 📈 **实时状态条** - 清晰显示各项数值
- ✨ **动画效果** - 喂食动画、爱心飘动

## 技术栈

- **Electron** - 桌面应用框架
- **Vue 3** - 前端框架
- **TypeScript** - 类型安全
- **Pinia** - 状态管理
- **Vite** - 构建工具

## 项目结构

```
Aemeath-AI/
├── src/
│   ├── main/                # Electron 主进程
│   │   ├── index.ts         # 入口文件

│   │   ├── App.vue          # 根组件
│   │   ├── components/      # 组件
│   │   ├── stores/          # Pinia 状态管理
│   │   ├── assets/          # 静态资源
│   │   └── styles/          # 样式文件
│   │
│   └── shared/              # 共享代码

│       └── defaults.ts      # 默认数据
│
├── data/                    # 运行时数据目录

└── tsconfig.json
```

## 开发指南

### 环境要求

- Node.js >= 16
- npm >= 8

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run electron:dev
```

这将启动 Vite 开发服务器并打开 Electron 窗口。

或者分开启动：

```bash
# 终端 1: 启动开发服务器
npm run dev

# 终端 2: 启动 Electron
electron .
```

### 构建

```bash
npm run build
```

这将构建项目并生成 Windows 安装包到 `release/` 目录。

## 功能演示

### 已实现功能

- ✅ **透明窗口** - 无边框、透明背景、置顶显示
- ✅ **窗口拖拽** - 按住鼠标拖动窗口到任意位置
- ✅ **喂食系统** - 右键菜单选择喂食，5 种食物可选
  - 🍎 苹果：+15 饱食，+5 好感
  - 🍰 蛋糕：+30 饱食，+15 好感
  - 🥛 牛奶：+10 饱食，+10 好感，+10 能量
  - 🍞 面包：+20 饱食，+3 好感
  - 🐟 烤鱼：+25 饱食，+20 好感
- ✅ **视觉反馈** - 喂食动画、爱心飘动、数值变化提示
- ✅ **状态系统** - 饱食度、好感度、能量实时显示
- ✅ **等级系统** - 喂食获得经验，自动升级
- ✅ **数据存储** - 自动保存到本地 JSON 文件

### 使用方法

1. **右键点击** 宠物打开菜单
2. 选择 **"喂食"** 打开食物选择界面
3. 点击任意食物进行喂食
4. 观察状态变化和动画效果

## 数据存储

数据使用本地 JSON 文件存储，位置在：

**Windows:** `C:\Users\<用户名>\AppData\Roaming\Aemeath-AI\data\`

数据文件：
- `pet.json` - 宠物状态数据
- `inventory.json` - 背包物品
- `settings.json` - 应用设置
- `feed_history.json` - 喂食记录
